#  Спецификация MVP: Telegram-бот для Amnezia VPN

## 1. Первый контакт и Онбординг (Вход в бот)

* **Экран до запуска бота (Описание):**
* Пользователь видит приветственный текст и кликабельную ссылку `[TG Proxy]` для обхода блокировок самого Telegram.


* **Команда `/start` (Первый запуск):**
* Бот отправляет приветствие с обращением по имени: *«Привет, [Имя]! Перед началом необходимо принять пользовательское соглашение»*.
* **Блокировка интерфейса:** Пользователь не получает доступа к главному меню, пока не примет условия.
* **Инлайн-кнопки под сообщением:**
* ` Прочитать соглашение` (открывает текст/ссылку).
* `Принять и продолжить` (записывает согласие в БД и открывает Главное меню).

---

##  2. Главное меню (Reply-клавиатура)

После принятия соглашения у пользователя внизу экрана навсегда закрепляется постоянная клавиатура управления:

```text
[ Профиль]       [ Оплата]
[ Подключение]   [ Поддержка]
[ Админка] (Видна только администраторам)

```

###  Раздел «Профиль»

Показывает текущий статус пользователя в системе.

* **Содержимое сообщения:**
* Данные аккаунта (Имя, Telegram ID).
* Статус подписки: **Активна до ДД.ММ.ГГГГ** или **Не активна**.
* Лимит устройств: **Использовано $X$ из $N$ слотов** (где $N$ — индивидуальный лимит пользователя).

###  Раздел «Оплата»

Интерфейс для покупки или продления подписки.

* **Содержимое сообщения:** Текст «Выбери период доступа:» и список тарифов.
* **Инлайн-кнопки (Тарифы):**
* `• n дней — X рублей / Stars`
* `• n дней — Y рублей / Stars`

* **Экран оплаты (после выбора тарифа):** Появляются две кнопки-заглушки для платежных систем:
* ` Telegram Stars`
* ` Platega`
(оплата пока что как заглушка при нажатии оплатить происходит успешная оплата)


###  Раздел «Подключение»

Основной хаб для управления VPN-профилями.

* **Если подписка не оплачена:** Бот пишет: *«Для доступа к серверам необходимо оплатить подписку»* и предлагает кнопку перехода в раздел оплаты.
* **Если подписка активна:**
* Бот показывает список уже созданных профилей (например: `🇳🇱 Нидерланды - iPhone`, `🇩🇪 Германия - PC`).
* Кнопка `➕ Создать профиль` (активна, если текущее число профилей меньше лимита $N$).


* **Процесс создания профиля:**
1. Нажатие на `➕ Создать профиль`.
2. Бот выводит инлайн-список доступных стран (серверов).
3. Пользователь выбирает страну.
4. Бот отправляет запрос к `amnezia-api`, регистрирует пользователя и открывает **Экран управления профилем**.


* **Экран управления конкретным профилем:**
* `🔑 Скопировать ключ` — 
* `📁 Скачать .conf` — 
* `🗑️ Удалить профиль` —

### Раздел «Поддержка»

* **Содержимое сообщения:** Контакты администратора/саппорта, гайд по установке на официальные приложения Amnezia VPN и короткая инструкция по настройке.

###  Раздел «Админка» (видно только для админа)

Доступен только пользователям, чьи Telegram ID внесены в список администраторов

* **Функционал:**
управление пользователями
управление серверами
статистика
рассылка

## ⚙️ 3. Подкапотная архитектура (Бэкенд и Логика)

* **Интеграция с Amnezia:** Бот общается с серверами через `amnezia-api`. При создании/удалении профиля бот шлет API-запросы на конкретный выбранный VPS.
* **Мульти-VPS:** Система поддерживает масштабирование. для запуска бота нужен как минимум 1 сервер, но архитектура БД позволяет в любой момент добавить второй, третий и последующие серверы. При создании профиля пользователь видит актуальный список всех рабочих серверов.
* **Гибкие лимиты:** По умолчанию каждому пользователю со старта дается базовый лимит на $N$ устройств (например, 3 устройства), но администратор может вручную через админку изменить этот параметр для конкретного ID (например, расширить до 5).
