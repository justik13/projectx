Ключевые факты: 
Протокол: ТОЛЬКО amneziawg2 (AmneziaWG 2.0)
API: kyoresuas/amnezia-api (REST-обёртка над Amnezia)

ЖЁСТКИЕ ПРАВИЛА (НАРУШЕНИЕ НЕДОПУСТИМО)
всегда смотреть документы в папке /docs 
1. Пользовательский опыт
Пользователю должно быть комфортно, удобно и интуитивно понятно
Балансировка ЗАПРЕЩЕНА — автоматический выбор сервера запрещён
Пользователь САМ выбирает сервер из списка при создании устройства
сейчас Реальных пользователей нет (миграции бд не нужны) но нужно быть готовым к реальным пользователям
2. Single Message Hub (SMH) 
Правила SMH:
исключение уведомления файлы и возможно другое
Если SMH невозможен например из за Telegram API — чётко говорить и предлагать решение

3. Абсолютная чистота чата
Если невозможно — создавать новое и удалять старое
Никаких тупиков — всегда есть путь назад
5. Запрещённые слова (в текстах БОТА/платежки которые доступны обычному пользователю)
VPN
впн
обход блокировок
глушилки
белые списки
LTE
6. Навигация
В КАЖДОЙ InlineKeyboard ОБЯЗАНА быть кнопка "Назад" или отмена в случаи платежей вести контекстно 
7. Команды бота
Разрешен ТОЛЬКО:
/start — запуск/перезапуск бота

Запрещены любые другие команды (/status, /profile, /devices и т.д.)

ФОРМАТЫ ОТВЕТОВ

Этап 1: Глубокий Анализ (запрещен поверхностный) 

1. Проведу полный глубокий аудит
2. Задам уточняющие вопросы ПЕРЕД генерацией файлов
3. Предложу несколько вариантов решения (если применимо) (также добавь простое обьяснение суть проблемы что будет если решить и тд)

Этап 2: Генерация кода 
(можно разбивать на несколько сообщений например если слишком много исправлений(10 файлов) отправить сначала 5 и потом сказать напишите далее чтобы отправить остальные  )
1. Выдам ПОЛНЫЕ файлы целиком (без сокращений)
2. Без "... остальной код без изменений"
3. Если нужно изменить несколько файлов — выдам все затронутые файлы

Этап 3: Проверка

1. Убедись что все жёсткие правила соблюдены

ПРИОРИТЕТЫ
1. Безопасность
2. Стабильность
3. UX — чистый чат, интуитивная навигация, понятные тексты
4. код — модульность без монолита читаемость
